		   一、游戏本体部分
1. 教学界面。
┌───────────────────────────────────────────────────────┐
　  与此同时，一份关于该游戏的教学将自动弹出并置顶在您的 　
　  所有窗口之上，请仔细观看四段教学视频，详细了解《求是 　
　  台球》的游玩方法。 					　
└───────────────────────────────────────────────────────┘

2. 游戏的主界面。
┌───────────────────────────────────────────────────────┐
　  当您关闭教学界面之后，便能看到《求是台球》的主界面， 　
　  在这里，通过点击玩家头像，可以分别进入两位玩家的个人 　
　  信息界面，并修改玩家头像、昵称、个人界面背景以及个人 　
　  对战时使用的球杆样式。				　
└───────────────────────────────────────────────────────┘

3. 游戏的控件栏。
┌───────────────────────────────────────────────────────┐
　  《求是台球》窗体的顶部是控件栏，除了最大化、最小化、 　
　  关闭之外，你还可以点击以下三个按钮：		   	　
　   （1）“小游戏机按钮”：进入游戏教程界面；		　
　   （2）“小齿轮按钮”：进入游戏设置界面；		　
　   （3）“信息按钮”：查看开发者信息。			　
　  游戏设置界面和教学界面可以在之后的任何时间自由进入。	　		
└───────────────────────────────────────────────────────┘

4. 游戏的玩法。
┌───────────────────────────────────────────────────────┐
　  《求是台球》提供了五个游戏模式，您可以选择任意一个游	　
　  戏模式进行游玩。进入游戏后，您可以按照以下方式进行游	　
　  戏的操作：						　
　    （1）鼠标左键移动：改变球杆方向；			　
　    （2）鼠标左键在球台上按住：开始蓄力；		　
　    （3）鼠标左键在球台上按住后松开：释放击球；	　
　    （4）鼠标左键在球台上按住后按下右键：取消蓄力；	　
　    （5）鼠标滚轮：调整下杆角度；			　
　    （6）鼠标左键下方中央目标球上拖动：改变击球点；	　
　    （7）键盘上、下、左、右键：改变击球点；		　
　    （8）自由球时鼠标左键：确定母球位置并摆放母球；	　
　    （9）键盘Esc键：进入暂停界面；			　
　  进入暂停界面，您可以保存或者退出当前游戏。		　	
└───────────────────────────────────────────────────────┘

5. 游戏的存档。
┌───────────────────────────────────────────────────────┐
　  您可以在游戏中进入暂停界面保存游戏，在存档管理界面当	　
　  中，您可以选择删除或者替换原有的存档，也可以将当前的	　
　  游戏进度保存在空的槽位上（共20个槽位）；您亦可以通过	　
　  在主界面点按“继续游戏”进入存档管理界面，读取之前的游	　
　  戏，或是删除过去的存档。

		   二、游戏文件部分
┌───────────────────────────────────────────────────────┐
　  下面简单介绍《求是台球》的目录下各文件的存储对象：	　
　   （1）Capt：在结束界面选择保存截图后的游戏截图。	　
　   （2）Data：游戏资源文件，切勿删除篡改，
　　　　　　　　但其中的 BGM 文件夹可以自行添加。 　
　   （3）Save：游戏存档文件，除测试外请勿擅自删除。	　
　   （4）User：游戏用户文件，任何情况请勿删除。		　
　   （5）Billiards.db：游戏图像素材文件，严禁删除。	　
　   （6）Sound.db：游戏音频素材文件，严禁删除。		　
　   （7）Config.ini：游戏配置文件，不建议手动修改。	　
　   （8）Billiards.exe：游戏程序可执行文件。
